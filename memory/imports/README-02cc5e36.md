# howto — рецепты сквозных изменений

Пошаговые плейбуки под частые задачи climem. Ценность — кодируют **связку слоёв**
(cli → command → store/embed/help), которую иначе переоткрываешь каждый раз.

| Рецепт | Когда |
|--------|-------|
| [add-command.md](add-command.md) | Новая подкоманда CLI (например `update`, `stats`) |
| [add-embedding-provider.md](add-embedding-provider.md) | Новый провайдер эмбеддингов (например локальный ONNX) |
| [add-import-format.md](add-import-format.md) | Новый формат документа для `import` |

> ⚠️ Имена символов проверены 2026-06-17. Номера строк не указываем — ориентируйся на имена
> функций/типов. Рецепт разошёлся с кодом → **почини рецепт** (часть работы, см.
> [../RULES.md](../RULES.md)).

Новый рецепт клади файлом `howto/<task>.md` и добавь строку в таблицу выше.
