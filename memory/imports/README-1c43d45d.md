# Заметки

Свободные заметки по фичам, компонентам и деталям реализации. Назначение — перед работой с
незнакомой областью прочитать соответствующую заметку, чтобы не переоткрывать уже
задокументированное.

| Файл | О чём |
|------|-------|
| [decisions.md](decisions.md) | Ключевые архитектурные решения и инварианты (направления, trade-offs) |

Заводи новый файл `notes/<тема>.md` под конкретную фичу/компонент, когда разобрался в
нетривиальной области. Связывай с [../known-issues.md](../known-issues.md),
[../structure.md](../structure.md) и (для глубоких разборов) `../analysis/`.
