Дефолты `cm init` без аргументов (2026-06-18): цель = текущая рабочая директория (корень проекта, неважно где лежит бинарь), имя папки = `memory` (было `.memory`), индексация кода ВКЛЮЧЕНА по умолчанию (`--no-code` отключает; `--code` больше не нужен). Резолв цели — чистый хелпер `init::resolve_target(p, cwd)` (первый позиционный → он, иначе cwd), вынесен чистым для тестируемости без мутации process-CWD. Так разворачивание = одна команда `cm init`: создаёт `<cwd>/memory/`, собирает доки (bulk-импорт .md, y/N) и строит граф кода. Связано: [[init-wire-entry-points]], граф кода `cm map`.
