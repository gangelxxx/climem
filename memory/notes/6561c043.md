`cm init` создаёт AGENTS.md если нет файла-инструкции (2026-06-18). Шаг [3/4]: если в корне НЕТ ни одного entry-point файла (CLAUDE/AGENTS/AGENT/GEMINI/.cursorrules/copilot-instructions) — создаётся AGENTS.md = AGENTS_HEADER (# AGENTS заголовок) + pointer-блок. Если есть хотя бы один — привязываем его, AGENTS.md не создаём. wire_entry_points возвращает WireResult{wired, any_existed}: any_existed отличает «файлов нет» от «есть, блок уже актуален» (иначе идемпотентный повтор ложно создал бы файл). create_agents_md не клоберит существующий. unwire_entry_points (deinit): если после снятия блока остался ровно AGENTS_HEADER — удаляет файл целиком ({"removed"}), round-trip fresh-проекта в ничто; пользовательский файл сохраняется. create_agents_md печатает {"wired":…,"new_file":true}. Связано: [[init-step-output]], [[init-no-arg-defaults]].
