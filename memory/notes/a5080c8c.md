Граф кода: фикс C# (2026-06-18). tree-sitter-c-sharp 0.23 tags.scm содержит лишнюю стансу `(namespace_declaration name: (identifier) @name) @module` с ГОЛЫМ @module — tree-sitter-tags 0.26 отвергает ВЕСЬ query при любом неизвестном capture, поэтому .cs не индексировался совсем (0 символов). Фикс: code::sanitize_tags_query режет стансу (разделены пустой строкой), если её последний классифицирующий @token не из ALLOWED_TAG_CAPTURES (@definition./@reference./@local./@name/@doc/@ignore). Применяется в parse() ко ВСЕМ грамматикам. Namespace остаётся через валидную @definition.module строку. ВАЖНО: init НЕ удаляет исходники — «Удалить исходные файлы? y» сносит только импортированные .md (копии в imports/, восстановимы); .cs читаются индексатором, не трогаются. Связано: [[init-docs-selection]].
